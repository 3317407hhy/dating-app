现在使用的是临时数据库，以后yxr或者ljj需要调用数据库都用这种方式


存数据：
// 把用户数据存进LocalStorage
localStorage.setItem('zx_user', JSON.stringify({
    name: "张三",
    email: "test@gmail.com",
    password: "123456",
    mbti: "INFP",
    interests: [],
    need: ""
}))


读数据：
// 取出来用
const user = JSON.parse(localStorage.getItem('zx_user'))
console.log(user.name) // 张三


删数据:
// 退出登录用
localStorage.removeItem('zx_user')
比如你想在某页面显示用户名:
//HTML
<span id="username"></span>
//JS
const user = JSON.parse(localStorage.getItem('zx_user'))
document.getElementById('username').innerText = user.name


显示其他字段一样的方式:
// 显示MBTI
document.getElementById('mbti').innerText = user.mbti

// 显示邮箱
document.getElementById('email').innerText = user.email

// 显示兴趣列表
document.getElementById('interests').innerText = user.interests.join('、')



核心逻辑就三步：

	1.	取数据 localStorage.getItem
	2.	找到页面元素 getElementById
	3.	把数据塞进去 innerText
